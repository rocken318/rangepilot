# 6max NLH Cash Game Preflop Range Matrix Tool

6maxキャッシュゲーム（ノーリミットテキサスホールデム）のプリフロップレンジを視覚的に確認できるツールです。

## 起動方法

```bash
npm install
npm run dev
```

ブラウザで http://localhost:5173 を開いてください。

## 機能

### モード
1. **オープンレンジ** - 各ポジションのオープンレンジ（5段階の幅で切替可能）
2. **vs オープンレイズ** - 前の人がオープンした時の対応（3ベット/コール/フォールド）
3. **vs 3ベット** - 自分のオープンに3ベットされた時の対応
4. **BBディフェンス** - BBから各ポジションのオープンに対する守り方
5. **SB vs BB** - SBオープンとBBディフェンスの両方
6. **相手タイプ別調整** - タイト/LAG/コーリングステーションなどへの調整ガイド
7. **メモ** - 自由なメモ（ローカルストレージに保存）

### エクスポート
- **PNG保存** - 現在のマトリックスを画像として保存
- **JSONコピー** - レンジデータをJSON形式でクリップボードにコピー
- **テキストコピー** - 簡易テキスト形式でコピー

## レンジデータの編集方法

レンジデータは `src/data/ranges.ts` に集約されています。

各シナリオのレンジは関数として定義されており、`setHands()` で各ハンドのアクションを設定しています。

```typescript
// 例: UTGのオープンレンジを編集
setHands(r, pp('5', 'A'), 'raise');  // 55+をレイズ
setHands(r, ['AKo','AQo'], 'raise'); // AKo, AQoをレイズ
setHands(r, ['A8s'], 'mixed', '状況次第');  // A8sをミックス（注釈付き）
```

ハンドの注釈は `HAND_NOTES` オブジェクトで管理しています。

## カスタマイズ

### レンジ幅の追加
`src/data/ranges.ts` の各 `make*Range` 関数内で、width パラメータに応じた分岐を追加してください。

### 色の変更
`src/components/HandMatrix.tsx` の `ACTION_COLORS` を編集してください。

### 新しいモードの追加
1. `src/types.ts` に新しいModeを追加
2. `src/data/ranges.ts` にレンジ生成関数を追加
3. `src/App.tsx` で新モードの表示ロジックを追加

## 技術スタック
- React + TypeScript
- Vite

## 注意事項
このレンジは6maxキャッシュゲーム向けの実戦用目安です。GTO完全解ではありません。相手のタイプ、レイズサイズ、スタック、アンティ有無によって調整してください。
